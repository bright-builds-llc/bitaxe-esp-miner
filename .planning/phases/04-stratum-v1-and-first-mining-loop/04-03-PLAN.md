## Trust Boundaries

| Boundary | Description |
| --- | --- |
| Stratum notify/extranonce -> BM1366 work fields | Pool-provided mining data becomes ASIC work only after typed parsing and hashing. |
| Clean-jobs signal -> valid-job tracking | Pool clean-jobs controls whether old ASIC nonces can be submitted. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
| --- | --- | --- | --- | --- |
| T-04-07 | Tampering | Mining job construction | mitigate | Decode hex and build typed fields with errors on malformed input; never zero-pad invalid data silently. |
| T-04-08 | Repudiation | Queue and valid-job tracker | mitigate | `clear_jobs` clears both queued work and valid job IDs so stale results are rejected. |
| T-04-09 | Denial of Service | Bounded work queue | mitigate | Capacity is fixed at upstream `QUEUE_SIZE` 12 and overflow returns `QueueFull` instead of unbounded allocation. |
</threat_model>
