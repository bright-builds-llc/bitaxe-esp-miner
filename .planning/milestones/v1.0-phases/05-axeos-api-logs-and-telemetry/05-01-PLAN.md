| Threat ID | Category | Component | Disposition | Mitigation Plan |
| --- | --- | --- | --- | --- |
| T-05-01 | Tampering | `wire.rs` DTOs | mitigate | Exact JSON fixture tests lock upstream field names, casing, units, and bool/number encodings. |
| T-05-02 | Information Disclosure | fixtures and tests | mitigate | Fixtures must use safe synthetic or public defaults only; no pool secrets or live logs. |
| T-05-03 | Repudiation | provenance | mitigate | Add reference breadcrumbs to module/test boundaries and keep `reference/esp-miner` read-only. |
| T-05-04 | Elevation of Privilege | public DTO contract | mitigate | DTOs expose safe blocked/unavailable hardware state and cannot enable Phase 6-owned control effects. |
